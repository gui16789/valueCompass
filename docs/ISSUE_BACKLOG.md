## Issue 1：完善模型配置中心

类型：feature

目标：

- 支持已保存模型提供商的编辑、测试、删除和角色模型配置。

当前状态：

- 已支持新增、编辑、删除、连接测试、TLS 高级开关。
- 测试连接已改为局部 fetch 更新。

后续范围：

- 每个 AI 角色单独设置模型名和温度。
- 支持设为默认提供商。
- 增加更清晰的连接错误分类。

验收标准：

- 用户可以为导师、反方委员、考官、研究助理分别设置模型。
- 角色配置保存后无需重建提供商。
- 测试失败时页面不整页刷新。
